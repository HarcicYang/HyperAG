## 食用方法

1. 克隆;
2. 参考[`HypeR Core`](https://github.com/HarcicYang/HyperBotCore)进行配置;
3. run

> HypeR Core 仅完整和稳定实现了 OneBot 11 协议，多协议支持尚不完备

## 已实现功能
- 群消息接收和发送
- 私聊消息接收和发送
- 历史记录存储
- 消息撤回
